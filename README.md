# Blockchain Voting DApp

This project lets people vote.
Votes are stored on blockchain.
You can see results in real time.

This project is made for beginners.
All code is simple and easy.

## Features
- Vote from web browser
- Read vote results
- Ethereum testnet support
- Simple user interface

## Requirements
- Node.js installed
- MetaMask wallet
- Testnet ETH
- RPC provider account

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add contract address and RPC URL
6. Run: npm start

## Example
Open browser.
Connect wallet.
Click vote button.

## Learning Goals
- Learn smart contracts
- Learn frontend Web3
- Learn blockchain state
